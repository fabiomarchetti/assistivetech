Questa cartella contiene la build pronta per il deploy su Aruba.

Carica tutto il contenuto via FTP nella directory agenda_timer del server Aruba.

Contenuto generato da flutter build web --base-href="/agenda_timer/".

Verifica che api/, assets/, icons/, index.html, manifest.json, main.dart.js, flutter.js, flutter_bootstrap.js, flutter_service_worker.js, favicon.png, version.json siano inclusi.
